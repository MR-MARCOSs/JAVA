A partir dos conceitos de classes, objetos, modificadores de acesso, crie um programa para uma biblioteca com os seguintes requisitos:

O sistema deve possibilitar o cadastro de livros
O sistema deve possibilitar o cadastro de leitores
O sistema deve possibilitar o cadastro de Temas de livros
O sistema deve possibilitar o empréstimo de livros
O sistema deve considerar a quantidade de exemplares de cada livro no momento do empréstimo (exemplo: se todos os exemplares do livro X estiverem emprestados, não deve ser possível haver novos empréstimos até que algum seja devolvido).
O sistema deve possibilitar a devolução de livros
Este programa deve possuir um menu e deve considerar a entrada dos usuários (dica: usar o Scanner para receber os valores do teclado). Um exemplo de menu é:
